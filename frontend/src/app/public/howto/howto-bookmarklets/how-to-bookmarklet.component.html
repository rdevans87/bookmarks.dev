Bookmarklets

Bookmarklets are little javascript links that live in the bookmarks toolbar of your browser. Drag one of the bookmarklet links bellow to your browser's bookmarks to easily save links and code snippets when clicked.

Bookmarks
  • - when clicked a popup to save the current URL is displayed [recommended for desktops and notebooks with good resolution]
  • - when clicked it redirects you to the New Bookmark form. Once added, you go back to the initial page [recommended for phones or smaller resolution devices]

You can also select the text on page that you want to be added to the description of the bookmark before you click the bookmarklet (this will have priority over the value from meta:description value which is used by default

You can also use the Save link to Codever extension for Chromium based browsers. It provides the same behaviour as the bookmarklet.

Snippets
  • - when clicked a popup to save the current URL is displayed [recommended for desktops and notebooks with good resolution]
  • - when clicked it redirects you to the New Snippet Form in the same window. Once added, you go back to the initial page [recommended for phones or smaller resolution devices]

You need to select the code snippet from the page you are surfing on. This will be "pasted" in the New Snippet Form
