Search

You can search bookmarks by placing your search terms in the Search bar. You can choose to search in Public Bookmarks, in My Bookmarks or My Snippets by selecting one of the options besides the in the Search Bar. When logged in, the default search domain is My Bookmarks and when not logged in, the default search domain is Public Bookmarks.

Tags must be embedded in square bracktes - e.g. [java]

!!! The search has an AND logic () as default, so adding a term will narrow your results, not expand them. !!!

You can use an OR logic, by adding the as query parameter in your search

Filters

You can filter further by adding the following parameters to your search
  • lang:iso_language_code - e.g. lang:en for English, lang:es for Spanish and lang:de for German bookmarks
  • site:site_URL - e.g. site:codepedia.org bookmarks only from website www.codepedia.org
  • private:only - when you want to search only through your private bookmarks
For example if you want to search for bookmarks that contain javascript, are in German and published on www.codepedia.org, you would use the following as query text in the search box: javascript lang:de site:codepedia.org

Save searches

You have the possibility to save this search by clicking the (Save search) button in the search bar. This way it will be displayed in the auto completion when you input text in the Search bar

Search via query parameters

You can execute the same search via query parameters

You can trigger the same search as above by calling the following URL: https://www.bookmarks.dev/search?q=exception [java]&sd=personal
  • q - here is your input search text
  • sd - search domain with two options
    • my-bookmarks - personal (private and public) bookmarks
    • public-bookmarks - public bookmarks (may include personal bookmarks if they come from same user)
    • my-snippets - personal code snippets

This is very handy if you use Web Search tools where you can define custom queries, like Alfred Web Search if you are on Mac

Import
Export bookmarks from favorite browsers
You can import your browser bookmarks to Codever. First you need to export the bookmarks as a html file from your favourite browser:
  • Chrome - Move bookmarks to another browser
  • Firefox - Export Firefox bookmarks to an HTML file to back up or transfer bookmarks
  • Safari - Import and export bookmarks in Safari on Mac
Import bookmarks to Codever
To import the bookmarks html file navigate to your Dashboard and select button under My Bookmarks
Note:
  • all imported browser bookmarks are private
  • all imported browser bookmarks are tagged with imported-browser plus folders they were found on (space characters in folder names are transformed in "-" - e.g. "Java tutorials" becomes "java-tutorials")
Export
You can export your personal bookmarks if you navigate to your Dashboard and select button under My Bookmarks
