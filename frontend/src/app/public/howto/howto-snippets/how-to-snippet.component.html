Code snippets

Code snippets are special types of bookmarks where you save one or more small pieces of code in the same record.

Save snippets
You can use the Code snippet option in right side of the menu bar
Bookmark Code snippet

or, for an easier experience use one of the following:

Browser Extension

We have browser extension to save code snippets for:
Chrome Firefox
Bookmarklet
Use our bookmarklet, if you can't or don't want to use Browser Extensions
The search at the end of the demo is configured with an AlfredApp custom query -
IntelliJ Plugin
If you use the IntelliJ Products use our - Save to Codever plugin
Please DO NOT store passwords or any sensitive information in your code snippets
Search snippets

Searching code snippets works the same as searching regular bookmarks. Just select the My Snippets option in the search bar:

You can save searches to reuse later the same way as for regular bookmarks.
